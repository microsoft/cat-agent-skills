# Voice Profile Template

Use this reference when the user has not supplied a complete voice guide. Fill only from evidence provided by the user or from clearly available source material.

## Voice profile

| Field | Guidance |
|---|---|
| Audience | Who the writing is for. |
| Region/language | Preferred language variant, spelling and terminology. |
| Tone | Practical, conversational, executive, technical, provocative, warm, etc. |
| Sentence rhythm | Short, medium, varied, reflective, direct, etc. |
| Paragraph style | Dense, spacious, short paragraphs, newsletter style, social post style, etc. |
| Preferred phrases | Terms the author often uses and wants retained. |
| Avoid list | Words, phrases, cliches or generic AI tells to remove. |
| Evidence rules | What the author is allowed to claim. |

## Common AI tells to remove

- In today's fast-paced world
- Unlock / unleash / harness the power of
- Game-changing / revolutionary / transformative without evidence
- Seamless / robust / cutting-edge as filler
- It is important to note
- Delve into / dive deep unless natural in context
- Overly neat three-part lists where the source did not require them
- Repetitive one-sentence paragraphs

## Rewrite test

Before returning the copy, ask:

1. Does this still mean what the user originally meant?
2. Has any unsupported claim been added?
3. Would this sound natural if read aloud?
4. Does the copy fit the channel and audience?
5. Has useful specificity been preserved rather than polished away?
