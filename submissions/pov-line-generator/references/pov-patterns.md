# POV Line Patterns

Use these patterns to create stronger point-of-view lines.

## Pattern 1: Common belief vs better belief

"The real issue is not [common belief]. It is [better belief]."

## Pattern 2: Activity vs outcome

"[Activity] only matters when it changes [outcome]."

## Pattern 3: Tooling vs operating model

"[Tool] adoption stalls when organisations treat it as technology rollout rather than operating model change."

## Pattern 4: Scaling tension

"You do not scale [capability] by adding more activity. You scale it by changing the system around it."

## Pattern 5: Executive reframing

"The question is no longer whether to do [thing]. It is whether the organisation can absorb the consequences of doing it well."

## Quality filters

A POV line is weak if it:

- Could apply to almost any topic.
- Uses hype instead of tension.
- Makes a claim that requires evidence not supplied.
- Sounds clever but does not help the audience think differently.
